## 🎨 自动排版特性

### 响应式网格布局

项目卡片采用**自适应网格系统**，会根据屏幕尺寸和项目数量自动排版：

```css
桌面端 (≥768px)：2列网格，每行显示 2 个项目
移动端 (<768px)：1列布局，垂直滚动
```

### 自动适配特性

✅ **自动换行** - 项目数量增加时自动创建新行
✅ **等高卡片** - 使用 Flexbox，所有卡片保持相同高度
✅ **间距统一** - Gap 属性确保所有卡片间距一致（24px）
✅ **图片裁剪** - 所有图片自动裁剪为 192px 高度，保持统一
✅ **对象适配** - `object-cover` 确保图片填充不变形

## 🔧 技术实现

### Tailwind CSS Grid

```html
<div class="grid md:grid-cols-2 gap-6">
  <!-- 项目卡片自动填充 -->
</div>
```

- `grid`: 启用 CSS Grid 布局
- `md:grid-cols-2`: 中等屏幕以上显示 2 列
- `gap-6`: 卡片间距 24px

### 自适应图片

```html
<img
  class="w-full h-full object-cover"
  :src="project.image"
/>
```

- `w-full h-full`: 填满容器
- `object-cover`: 保持比例裁剪（不变形）

### 等高卡片

```html
<GlassCard class="h-full flex flex-col">
  <!-- 内容 -->
</GlassCard>
```

- `h-full`: 撑满网格单元格高度
- `flex flex-col`: 内容垂直排列

## 🎨 图片优化建议

### 推荐规格
- **尺寸**: 1200×800px 或 16:9 比例
- **格式**: PNG（有透明）或 JPG（照片）
- **大小**: < 500KB（已压缩）
- **命名**: 小写字母，无空格，如 `project-name.png`

### 批量优化工具

**在线工具：**
- [TinyPNG](https://tinypng.com/) - PNG 无损压缩
- [Squoosh](https://squoosh.app/) - 多格式压缩

**命令行工具：**
```bash
# 安装 ImageMagick
# 批量调整尺寸
magick mogrify -resize 1200x800^ -gravity center -extent 1200x800 *.png

# 批量压缩
magick mogrify -quality 85 *.jpg
```

## ✨ 高级功能

### 1. 限制显示数量

如果只想显示前 N 个项目：

```typescript
// Projects.vue
<ProjectCard
  v-for="project in data.projects.slice(0, 6)"  // 只显示前6个
  :key="project.id"
  ...
/>
```

### 2. 排序项目

按时间倒序排列（最新的在前）：

```typescript
// resume.ts
projects: [
  { period: '2025.04 - 2025.08', ... },  // 最新
  { period: '2025.01 - 2025.03', ... },
  { period: '2024.10 - 2024.12', ... },
  // ...
]
```

### 3. 分类显示

如果项目很多，可以按类型分组：

```typescript
// 可以添加 category 字段
{
  id: 'proj-1',
  category: '图像处理',  // 新增字段
  title: '唐卡图像去噪系统',
  // ...
}
```
