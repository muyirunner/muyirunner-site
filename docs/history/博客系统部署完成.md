## 🚀 如何添加新博客文章？

### 方式 1：直接在本地添加

1. **创建 Markdown 文件**

```bash
# 在 public/blog/posts/ 目录创建新文章
# 文件名格式: your-post-title.md
```

示例内容：

```markdown
# 我的新文章标题

这是文章的第一段...

## 二级标题

内容...

​```javascript
// 代码示例
console.log('Hello World')
​```
```

2. **更新文章索引**

编辑 `public/blog/posts.json`，添加新文章的元数据：

```json
{
  "posts": [
    {
      "id": "your-post-title",
      "title": "我的新文章标题",
      "description": "文章简介",
      "author": "杨汶川",
      "date": "2025-12-26",
      "category": "技术",
      "tags": ["Vue", "前端"],
      "readTime": 5,
      "cover": "/blog/images/cover.jpg",
      "file": "/blog/posts/your-post-title.md"
    },
    // ... 其他文章
  ]
}
```

3. **构建并部署**

```bash
# 本地构建
npm run build

# 部署到服务器
scp -r dist/* admin@muyirunner.icu:/var/www/html/
```

### 方式 2：直接在服务器上添加 ⭐ 推荐

```bash
# SSH 登录服务器
ssh admin@muyirunner.icu

# 创建新文章
sudo nano /var/www/html/blog/posts/my-new-post.md
# 编写内容，保存

# 更新索引
sudo nano /var/www/html/blog/posts.json
# 添加新文章元数据，保存

# 完成！立即生效
```

## 🎨 系统特点

### ✅ 优势

1. **零内存占用**：纯静态文件，不需要任何后端进程
2. **极致性能**：所有内容都是静态的，加载速度飞快
3. **易于维护**：Markdown 写作，Git 版本控制
4. **安全可靠**：没有数据库，无需担心被攻击
5. **支持代码高亮**：使用 highlight.js
6. **响应式设计**：完美适配手机/平板/电脑

### 📊 技术栈

- **前端框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **Markdown 解析**: marked.js
- **代码高亮**: highlight.js
- **样式**: Tailwind CSS
- **部署**: 静态文件服务器 (Nginx)

## ⚠️ 注意事项

1. **文件权限**：确保服务器上的文件有正确的权限
   ```bash
   sudo chown -R admin:admin /var/www/html/blog
   sudo chmod -R 755 /var/www/html/blog
   ```

2. **图片优化**：博客封面图建议压缩到 < 200KB

3. **备份**：定期备份 `public/blog/` 目录，或使用 Git 管理

4. **Swap 空间**：已为你的服务器添加了 1GB swap，开机会自动启用
