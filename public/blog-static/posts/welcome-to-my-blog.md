## 关于这个网站

这个网站是我用 **Vue 3 + TypeScript + Vite** 构建的个人主页和博客。选择这套技术栈的原因是：

- ✨ **现代化**：Vue 3 Composition API 让代码更加简洁优雅
- ⚡ **高性能**：Vite 提供了极快的开发体验
- 🎨 **可定制**：完全掌控每一个设计细节
- 📱 **响应式**：完美适配各种设备

## 为什么选择静态博客？

相比传统的数据库驱动的 CMS 系统，静态博客有很多优势：

1. **极致性能**：所有内容都是静态文件，加载速度飞快
2. **零成本维护**：不需要数据库，不需要后端服务器
3. **安全可靠**：没有数据库被攻击的风险
4. **版本控制**：博客文章用 Markdown 写，可以用 Git 管理
5. **低资源占用**：非常适合我的低配服务器（407MB 内存）😄
